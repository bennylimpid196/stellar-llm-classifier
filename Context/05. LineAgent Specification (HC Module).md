## 1. Overview

El `LineAgent` es un módulo de Hard Computing (HC) responsable de extraer parámetros físicos de líneas espectrales específicas, principalmente el Triplete de Calcio (CaT) en la ventana RVS o H-alpha. Recibe un espectro normalizado y ajusta un perfil de Voigt para cuantificar el Ancho Equivalente (EW), el FWHM total y los desplazamientos del centroide (velocidad Doppler). El módulo incluye priors físicos restrictivos basados en la resolución instrumental de Gaia RVS para estabilizar convergencias.

## 2. Metodología Matemática

### 2.1 Ajuste de Perfil (Voigt1D) y Limites Físicos (Bounds)

1. **Inicialización y Prior Instrumental:** El agente aísla una ventana local alrededor de la longitud de onda de reposo. El modelo se inicializa utilizando la resolución instrumental del espectrógrafo RVS ($R \sim 11500$, equivalente a $\Delta\lambda \sim 0.075$ nm a 860 nm).
    
2. **Límites Estrictos:** * El ensanchamiento Gaussiano (`fwhm_G`) no puede ser inferior a la resolución instrumental.
    
    - El ensanchamiento Lorentziano (`fwhm_L`) debe ser estrictamente positivo.
        
    - La amplitud se acota a `(-0.5, 1.2)` (recordando que el espectro de absorción se invierte para el ajuste).
        
3. **Ancho Equivalente (EW) Continuo:** Para calcular el área sin trucar las alas, se genera una grilla fina extendida, se evalúa el modelo, se eliminan numéricamente las oscilaciones no físicas (`np.clip(..., 0.0, None)`) y se integra de forma continua garantizando la estabilidad de `np.trapz`.
    

## 3. Banderas Lógicas (Predigestión)

- **`high_quality_fit`**: `True` si el algoritmo Levenberg-Marquardt convergió con éxito y el centroide ajustado está dentro de límites astrofísicos realistas.
    
- **`is_broad_line`**: `True` si el FWHM total supera los 0.5 nm (indicador de rotaciones rápidas o presiones extremas).
    
- **`has_emission_core`**: `True` si el valor del modelo evaluado en el centroide invertido es negativo.
    

## 4. Plantilla de Implementación (Python)

```python
import numpy as np
import logging
import warnings
from typing import Dict, Any, Tuple
from astropy.modeling import models, fitting
from astropy.utils.exceptions import AstropyUserWarning

logger = logging.getLogger(__name__)

# Instrumental resolution prior for Gaia RVS (R ~ 11500 at 860 nm)
FWHM_INSTRUMENTAL_RVS_NM = 0.075

class LineAgent:
    """
    HC Module to fit physical Voigt profiles to normalized spectral lines.
    """

    def __init__(self, rest_wavelength: float, window_size_nm: float = 1.5):
        self.rest_wavelength = rest_wavelength
        self.window_size_nm = window_size_nm
        self.fitter = fitting.LevMarLSQFitter()

    def _isolate_line_window(self, wave: np.ndarray, flux: np.ndarray) -> Tuple[np.ndarray, np.ndarray]:
        """Extracts the local spectral window around the rest wavelength."""
        mask = (wave >= self.rest_wavelength - self.window_size_nm) & \
               (wave <= self.rest_wavelength + self.window_size_nm)
        return wave[mask], flux[mask]

    def process(self, wavelength: np.ndarray, normalized_flux: np.ndarray, source_id: str = "UNKNOWN") -> Dict[str, Any]:
        """Executes the Levenberg-Marquardt fit on the provided spectral array."""
        res = {
            "module": "LineAgent",
            "status": "failed",
            "target_line_nm": self.rest_wavelength,
            "measurements": {
                "equivalent_width_aa": np.nan,
                "fwhm_total_nm": np.nan,
                "centroid_shift_nm": np.nan
            },
            "logical_flags": {
                "high_quality_fit": False,
                "is_broad_line": False,
                "has_emission_core": False
            }
        }

        try:
            x_window, y_window = self._isolate_line_window(wavelength, normalized_flux)

            if len(x_window) < 7:
                logger.error(f"[{source_id}] Insufficient points ({len(x_window)}) for line {self.rest_wavelength} nm.")
                return res

            # Model Initialization (Inverted for absorption)
            y_inverted = 1.0 - y_window
            amp_guess = max(float(np.max(y_inverted)), 0.05)
            
            voigt_init = models.Voigt1D(
                x_0=self.rest_wavelength,
                amplitude_L=amp_guess,
                fwhm_L=0.05, 
                fwhm_G=FWHM_INSTRUMENTAL_RVS_NM
            )
            
            # Physical Bounding
            voigt_init.x_0.bounds = (self.rest_wavelength - 1.5, self.rest_wavelength + 1.5)
            voigt_init.amplitude_L.bounds = (-0.5, 1.2)
            voigt_init.fwhm_G.bounds = (FWHM_INSTRUMENTAL_RVS_NM, None) # Cannot be sharper than instrument
            voigt_init.fwhm_L.bounds = (0.0, None)                      # Cannot be negative

            # Perform Fit
            with warnings.catch_warnings(record=True) as caught_warnings:
                warnings.simplefilter("always", AstropyUserWarning)
                fitted_model = self.fitter(voigt_init, x_window, y_inverted)
                for w in caught_warnings:
                    logger.debug(f"[{source_id}] Astropy warning at {self.rest_wavelength} nm: {w.message}")

            fit_info = self.fitter.fit_info
            if fit_info['ierr'] not in [1, 2, 3, 4]:
                logger.warning(f"[{source_id}] Fit failed to converge for line {self.rest_wavelength}.")
                return res

            # EW calculation over a continuous fine grid
            x_fine = np.linspace(self.rest_wavelength - 3.0, self.rest_wavelength + 3.0, 1000)
            fitted_fine = fitted_model(x_fine)
            
            # Clip non-physical negative wings to 0.0 to prevent integrating noise
            fitted_clipped = np.clip(fitted_fine, 0.0, None)
            
            ew_nm = float(np.trapz(fitted_clipped, x_fine))
            ew_aa = ew_nm * 10.0  # Convert to Angstroms
            
            # FWHM total (Thompson et al. 1987)
            f_l = fitted_model.fwhm_L.value
            f_g = fitted_model.fwhm_G.value
            fwhm_total = float(0.5346 * f_l + np.sqrt(0.2166 * f_l**2 + f_g**2))
            centroid_shift = float(fitted_model.x_0.value - self.rest_wavelength)
            
            # Emission core check
            peak_value = float(fitted_model(fitted_model.x_0.value))
            is_emission = bool(peak_value < 0)

            # Populate Contract
            res["measurements"] = {
                "equivalent_width_aa": round(ew_aa, 3),
                "fwhm_total_nm": round(fwhm_total, 4),
                "centroid_shift_nm": round(centroid_shift, 4)
            }
            res["logical_flags"] = {
                "high_quality_fit": True,
                "is_broad_line": bool(fwhm_total > 0.5),
                "has_emission_core": is_emission
            }
            res["status"] = "success"

        except Exception as e:
            logger.error(f"[{source_id}] LineAgent critical failure: {e}")

        return res
```
