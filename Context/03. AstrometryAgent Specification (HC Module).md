## 1. Overview

The `AstrometryAgent` is a Hard Computing (HC) module responsible for calculating distance-dependent physical properties and geometric kinematics. It transforms raw Gaia astrometry and photometry into intrinsic stellar parameters (Absolute Magnitude) and dynamic indicators (Tangential Velocity). It performs strict "Logical Predigestion" by generating boolean flags that protect the Soft Computing (SC) layer from evaluating mathematically invalid or highly uncertain data.

## 2. Mathematical Formulations

### 2.1 Distance and Absolute Magnitude

The agent calculates the Absolute Magnitude in the G-band ($M_G$). It strictly requires a positive parallax ($\varpi > 0$).

1. **Extinction Correction ($A_G$):** Using the updated coefficient for Gaia DR3 (Cardelli / Wang & Chen 2019):
    
    $$A_G=2.74 \cdot E(BP-RP)$$
    
2. **Absolute Magnitude ($M_G$):**
    
    $$M_G=G+5+5\log_{10}(\varpi/1000)-A_G$$

### 2.2 Tangential Velocity ($V_{tan}$)

Calculates the velocity perpendicular to the line of sight (in km/s). Note that in Gaia DR3, the `pmra` field is already corrected by the declination cosine ($\mu_{\alpha*} = \mu_\alpha \cos\delta$). Thus, the total proper motion is strictly $\sqrt{pmra^2 + pmdec^2}$.

$$V_{tan}=4.74\frac{\sqrt{\mu_{\alpha*}^2+\mu_\delta^2}}{\varpi}$$

## 3. Logical Flags (Predigestion Logic)

The agent evaluates and exports the following booleans to the JSON contract. Dependent flags strictly require astrometric reliability to prevent propagating noise.

- **`is_reliable_parallax`**: `True` if `parallax / parallax_error > 5`.
- **`is_giant`**: `True` if `abs_mag < 3.0` AND `teff_k < 7000` AND `is_reliable_parallax == True`.
- **`is_high_velocity`**: `True` if `v_tan > 200` AND `is_reliable_parallax == True`.
- **`is_binary_candidate`**: `True` if `ruwe > 1.4` (Evaluated independently of parallax validity).

## 4. Input Requirements

The module requires a dictionary or DataFrame row containing:

- `parallax`, `parallax_error`, `phot_g_mean_mag`, `ebpminrp_gspphot`, `pmra`, `pmdec`, `ruwe`, `teff_gspphot`.
## 5. Implementation Template (Python)


```python
import numpy as np
import logging
from typing import Dict, Any, Optional

logger = logging.getLogger(__name__)

# Standard coefficient for Gaia G-band according to Wang & Chen (2019)
K_G_DEFAULT = 2.74

class AstrometryAgent:
    """
    HC Module to derive intrinsic stellar properties from Gaia astrometry.

    Notes:
        - pmra in Gaia DR3 is already mu_alpha* (cos-delta corrected), so
          total proper motion is strictly sqrt(pmra² + pmdec²).
        - Distance-dependent measurements (abs_mag, v_tan) are only exported
          when is_reliable_parallax is True, to prevent the SC layer from
          receiving physically absurd values.
        - teff_gspphot is passed through to measurements so the SC layer
          has it available in a single contract without querying other modules.
    """

    def __init__(self, extinction_coeff: float = K_G_DEFAULT):
        self.k_g = extinction_coeff

    def process(self, data: Dict[str, Any]) -> Dict[str, Any]:
        """
        Calculates absolute magnitude, tangential velocity, and logical flags.

        Guards applied:
            1. Negative/zero parallax  → early return with status 'partial'.
            2. NaN in ebpminrp_gspphot → A_G set to 0.0 with warning.
            3. Unreliable parallax     → distance measurements exported as None.
        """
        res = {
            "module": "AstrometryAgent",
            "status": "failed",
            "measurements": {
                "absolute_magnitude_g":     None,
                "tangential_velocity_km_s": None,
                "extinction_ag":            None,
                "teff_k":                   None
            },
            "logical_flags": {
                "is_reliable_parallax": False,
                "is_giant":             False,
                "is_high_velocity":     False,
                "is_binary_candidate":  False
            }
        }

        source_id = data.get('source_id', 'UNKNOWN')

        try:
            # --- Extract inputs ---
            p      = data['parallax']
            p_err  = data['parallax_error']
            g_mag  = data['phot_g_mean_mag']
            ebp_rp = data['ebpminrp_gspphot']
            teff   = data['teff_gspphot']
            ruwe   = data['ruwe']
            pm     = np.sqrt(data['pmra']**2 + data['pmdec']**2)

            # teff is always exportable regardless of parallax quality
            res["measurements"]["teff_k"] = int(teff) if not np.isnan(teff) else None

            # --- Guard 1: Negative or zero parallax ---
            # log10 is undefined; distance and velocity are meaningless.
            if p <= 0:
                logger.warning(
                    f"[{source_id}] Negative/zero parallax ({p} mas). "
                    "Distance-dependent quantities not computed."
                )
                res["logical_flags"]["is_binary_candidate"] = bool(ruwe > 1.4)
                res["status"] = "partial"
                return res

            # --- Reliability check ---
            # Must be evaluated after the p > 0 guard to avoid ZeroDivisionError.
            is_reliable = bool((p / p_err) > 5)

            # --- Guard 2: NaN extinction ---
            # GSP-Phot may not converge for very hot or very cool stars.
            # Conservative fallback: assume zero extinction rather than propagate NaN.
            if np.isnan(ebp_rp):
                logger.warning(
                    f"[{source_id}] ebpminrp_gspphot is NaN. "
                    "Setting A_G = 0.0 (no extinction correction applied)."
                )
                ebp_rp = 0.0

            # --- Physical calculations ---
            a_g     = self.k_g * ebp_rp
            abs_mag = g_mag + 5 + 5 * np.log10(p / 1000) - a_g
            v_tan   = 4.74 * pm / p

            # --- Guard 3: Suppress unreliable distance measurements ---
            # A low parallax SNR can produce valid-looking but physically absurd
            # values (e.g. v_tan > 10^6 km/s). Export None instead of noise.
            res["measurements"]["extinction_ag"] = round(float(a_g), 4)

            if is_reliable:
                res["measurements"]["absolute_magnitude_g"]     = round(float(abs_mag), 4)
                res["measurements"]["tangential_velocity_km_s"] = round(float(v_tan), 4)
            else:
                logger.info(
                    f"[{source_id}] Parallax SNR = {p/p_err:.2f} < 5. "
                    "abs_mag and v_tan suppressed from contract."
                )

            # --- Logical Predigestion ---
            # is_giant requires is_reliable: abs_mag without reliable parallax
            # has no physical meaning and must not influence classification.
            # is_high_velocity inherits the same guard via is_reliable.
            res["logical_flags"] = {
                "is_reliable_parallax": is_reliable,
                "is_giant":             bool(abs_mag < 3.0 and teff < 7000 and is_reliable),
                "is_high_velocity":     bool(v_tan > 200 and is_reliable),
                "is_binary_candidate":  bool(ruwe > 1.4)  # Independent of distance
            }
            res["status"] = "success"

        except KeyError as e:
            logger.error(f"[{source_id}] Missing required input field: {e}")
        except ZeroDivisionError:
            logger.error(f"[{source_id}] parallax_error is zero. Cannot compute SNR.")
        except Exception as e:
            logger.error(f"[{source_id}] AstrometryAgent failed unexpectedly: {e}")

        return res
```
